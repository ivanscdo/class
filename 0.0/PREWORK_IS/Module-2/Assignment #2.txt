Scrum - framework for managing software development work with an emphasis on a fast pace and delegation.

SOAP - Simple Object Acess Protocol; legacy protocal created by Microsoft, used to access to web services.

REST - Representational State Transfer; an architecture style that is a lighter weight alternative to SOAP.

JSON - lightweight data-interchange format based on subset of JavaScript Programming Language.

jQuery - Javascript library that makes certain processes simpler with an easy-to-use API that works across several browsers. 

API - Application Programing Interface; set of tourtines, protocla,s and tools for building applications, specifies interactions between software components

LAMP - Linux, Apache, MySQL, PHP; open-source web development platform that uses Linux as the operating system, Apache as the Web server, MySQL as the RDBMS and PHP as the object-oriented scripting language.

AngularJS - JavaScript-based open-source front-end web application framework mainly maintained by Google

React - JavaScript library for building user interfaces, maintained by Facebook, Instagram and a community of individual developers and corporations.


Backbone - JavaScript library with a RESTful JSON interface and is based on the model–view–presenter application design paradigm.
